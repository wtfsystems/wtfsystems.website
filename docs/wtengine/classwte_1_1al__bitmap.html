wte::al_bitmap::al_bitmap ( const bool &  p)

Set bitmap preservation.

Set the nopreserve flag to true to create a bitmap that does not preserve its data when the screen is reset. This is to increase performance when drawing. See Allegro docs on graphics routines for more info.

Parameters
pPreservation flag.
